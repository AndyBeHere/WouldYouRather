# Would You Rather

# Installing

Fork and clone this repository. Then, using npm:

```bash
$ npm install
```

# Launching

Project can be launched in dev mode by 

```bash
$ npm start
```

or build it with 

```bash
npm run build
```

and deploy **build** folder, for example

```bash
npx serve build
```
